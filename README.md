# React Tasks

This repository contains two separate React apps:

```
task1/
task2/
```

---

## ▶️ Run Task 1

```bash
cd task1
npm i
npm run dev
```

---

## ▶️ Run Task 2

```bash
cd task2
npm i
npm run dev
```

---

## 👀 Preview

After running `npm run dev`, open the local URL shown in the terminal (usually http://localhost:5173).
